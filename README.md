# Система управления турнирами

## Описание проекта

Система управления турнирами - это веб-приложение, разработанное на React и TypeScript, которое позволяет организовывать и проводить турниры в формате плей-офф. Приложение предоставляет возможности для настройки параметров турнира, регистрации участников, проведения жеребьевки, ввода результатов матчей и отображения турнирной сетки и таблицы результатов.


## Функциональность

Основные возможности приложения:

- **Настройка турнира:**
    - Задание названия турнира
    - Выбор количества участников (4, 8, 16, 32)
    - Настройка количества партий на каждом этапе
    - Выбор типа тайбрейка (разница голов, серия пенальти, дополнительное время)
    - Выбор типа сетки (жесткая сетка, перепосев, жеребьевка после каждого тура)

- **Управление участниками:**
    - Добавление данных участников (имя, фамилия, команда, город, рейтинг)
    - Редактирование и удаление участников
    - Автоматическое распределение посева на основе рейтинга

- **Турнирная сетка:**
    - Визуализация турнирной сетки с соединительными линиями
    - Отображение текущего раунда
    - Отображение результатов матчей
    - Автоматическое продвижение победителей в следующие раунды

- **Управление матчами:**
    - Ввод результатов матчей
    - Автоматическое определение победителей
    - Переход к следующему раунду после завершения всех текущих матчей

- **Таблица результатов:**
    - Сводная статистика по всем участникам
    - Детальный просмотр результатов каждого участника
    - Сортировка по различным показателям

## Технологии

- **Frontend:**
    - React 18.x
    - TypeScript 4.x
    - React Router для навигации
    - Context API для управления состоянием

- **Стилизация:**
    - CSS Modules для компонентного подхода к стилям
    - Адаптивный дизайн для корректного отображения на различных устройствах

- **Сборка:**
    - Create React App для настройки и сборки приложения

## Установка и запуск

### Предварительные требования

- Node.js (v16.x или выше)
- npm (v8.x или выше) или yarn (v1.22.x или выше)

### Шаги по установке

1. Установите зависимости:
   ```
   npm install
   ```
   или
   ```
   yarn install
   ```

2. Запустите приложение в режиме разработки:
   ```
   npm start
   ```
   или
   ```
   yarn start
   ```

3. Откройте [http://localhost:3000](http://localhost:3000) в вашем браузере.

### Сборка для продакшн

Для создания оптимизированной сборки выполните:
```
npm run build
```
или
```
yarn build
```

## Использование системы

### Настройка турнира

1. На начальной странице заполните форму настроек турнира:
    - Введите название турнира
    - Выберите количество участников
    - Укажите количество партий на каждом этапе
    - Выберите тип тайбрейка
    - Выберите тип сетки

2. Нажмите кнопку "Далее: Добавить участников"

### Добавление участников

1. Заполните форму для каждого участника:
    - Имя и фамилия
    - Название команды (опционально)
    - Город (опционально)
    - Рейтинг

2. Нажмите кнопку "Добавить" для каждого участника

3. После добавления необходимого количества участников (минимум 2) нажмите "Сформировать турнирную сетку"

### Работа с турнирной сеткой

1. На странице отобразится сгенерированная турнирная сетка
2. Матчи текущего раунда будут выделены
3. Щелкните на матч, чтобы ввести результат
4. После ввода результатов всех матчей текущего раунда станет активной кнопка "Следующий раунд"
5. Нажмите кнопку для перехода к следующему раунду

### Ввод результатов матча

1. На странице матча вы увидите информацию об участниках
2. Введите счет для обоих участников
3. Нажмите "Сохранить результат"
4. Вы вернетесь к турнирной сетке с обновленными данными

### Таблица результатов

1. Нажмите кнопку "Таблица результатов" на странице турнирной сетки
2. Вы увидите сводную таблицу с результатами всех участников
3. Щелкните на строку участника для просмотра подробной информации

## Структура проекта

```
tournament-bracket/
├── public/               # Статические файлы
├── src/                  # Исходный код
│   ├── assets/           # Изображения и иконки
│   ├── components/       # React компоненты
│   ├── context/          # Context API для управления состоянием
│   ├── types/            # TypeScript типы и интерфейсы
│   ├── App.tsx           # Главный компонент приложения
│   ├── index.tsx         # Точка входа в приложение
│   └── index.css         # Глобальные стили
├── package.json          # Зависимости и скрипты
└── tsconfig.json         # Конфигурация TypeScript
```

### Основные компоненты

- **TournamentSettings**: Настройка параметров турнира
- **ParticipantsList**: Управление списком участников
- **TournamentBracket**: Визуализация турнирной сетки
- **MatchDetails**: Ввод результатов матча
- **ResultsTable**: Отображение таблицы результатов

### Контекст

- **TournamentContext**: Управление данными турнира и бизнес-логикой
